Introduction

This is a design study!

This design study of a next generation Paper UI is aiming to solve these issues:

  • Modern, maintainable technology stack for easy extensions.
  • A newby friendly presentation, including a features tour and a guided tutorial on how to begin with openHAB.
  • Covering all openHAB functionality, including Item Tags and Automation Rules, so that falling back to text files or the command line should not be necessary at any time.
  • Efficient multi item / batch editing features via textual view mode and multi-item selection.
